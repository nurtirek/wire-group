04 Inside Call Analytics

Every call worth hearing, scored and sorted into one screen.

01 Daily summary

Day summary — the floor in one screen

Average score for the day, how many calls were good, average and bad, and how many interested clients were lost. The whole shift, at a glance.

  • Average score 0–10
  • Good / average / bad split
  • Lost-deal counter
02 Triage

Only what's worth your time

At the end of the day the list keeps just the calls you should actually hear — the single best one and every bad or lost-deal call. The rest is hidden.

  • Best call + all problem calls
  • Reason to listen on every card
  • No more scrolling 200 calls
03 Operators

Every call tied to its operator

The operator's name comes straight from Binotel and sits on every call. A per-operator board shows the weakest on top and crowns the best of the day.

  • Name on each call (from Binotel)
  • Average score per operator
  • Best operator of the day
04 Killer feature

Recording to score — fully automatic

The bot pulls the recording from Binotel, transcribes it locally with Whisper (audio never leaves the server), and Claude scores the operator 0–10 with a written breakdown. No one presses play.

  • Local Whisper transcription · ru / kk
  • Claude scores script, politeness, closing
  • Summary, client need and objections
  • "Next step secured" — yes / no
05 Lost deals

Lost-deal radar

The AI catches the worst miss in sales: a client who was clearly interested, but the operator never closed on a next step. Those calls get flagged so you can coach and call back.

  • Interest level read from the talk
  • Next-step check, yes / no
  • Flag on every missed deal
06 Auto-triage

Smart nightly triage

You don't analyse 300 calls a day. Each night the bot picks the suspicious ones — very short calls, clients who called back, plus a fair sample per operator — within a daily limit. Cheap, and it still catches the problems.

  • Runs automatically every night
  • Short · repeat · per-operator sample
  • Daily limit keeps cost in check
05 Onboarding

Two keys. Under a minute.

No installs, no app store. Call Analytics lives inside Telegram and connects to Binotel with two API keys.

  1. 01

    Open the bot

    Launch @binotelanalyzebot in Telegram and tap to open the Mini App.

  2. 02

    Binotel Key + Secret

    Paste your Binotel API Key and Secret (from your Binotel cabinet → API).

  3. 03

    That's it

    The bot analyses last night's calls automatically. Open "Yesterday" and read the summary.

Privacy · the details

  • Audio is transcribed locally with Whisper — the recording never leaves the server for a third-party transcription service.
  • Binotel keys are stored encrypted (Fernet). The only paid external API is Anthropic, for scoring the text.
  • Mini App auth via Telegram initData (HMAC signature). No separate passwords.
  • Backend in FastAPI on Render, integration with Binotel API 4.0.
